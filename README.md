# ResumeAI 🚀  
An AI-powered Resume Analyzer that evaluates resumes, calculates ATS score, and provides intelligent improvement suggestions.

---

## 📌 Overview
ResumeAI helps job seekers optimize their resumes by analyzing:
- Keyword relevance
- Skills match
- ATS compatibility
- Content quality

It generates actionable suggestions to improve selection chances in product-based companies.

## ⚙️ How to Run Locally

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Ayush4511/ResumeAI.git
cd ResumeAI
```

### 2️⃣ Setup Environment and Keys
Create a `.env` file in the root directory and add your API keys:
```env
GEMINI_API_KEY=your_gemini_api_key
GOOGLE_CLIENT_ID=your_google_oauth_client_id
GOOGLE_CLIENT_SECRET=your_google_oauth_client_secret
```

### 3️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application
```bash
python -m flask run
```
